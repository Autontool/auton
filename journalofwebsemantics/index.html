Questionnaire

Below we list the questions used for the evaluation section in the paper

  1. Are you familiar with software engineering?
  2. Are you familiar with ontology engineering?
  3. How many ontologies have you built?
  4. I found OnToology useful for supporting ontology engineering tasks?
  5. I found OnToology useful for supporting collaborative engineering tasks?
  6. OnToology saves me time when I want to create the documentation of an ontology?
  7. OnToology saves me time when I want to evaluate an ontology
  8. OnToology saves me time when I want to publish an ontology
  9. OnToology saves me time when I want to visualize an ontology
  10. It is easy to use
  11. I can use it without written instructions
  12. I can recover from mistakes quickly and easily
  13. I can use OnToology successfully
  14. I learned to use OnToology quickly
  15. I am satisfied with OnToology's visualizations
  16. I am satisfied with OnToology's evaluation
  17. I am satisfied with the documentation generated by OnToology
  18. I am satisfied with the way OnToology published my ontology
  19. I would recommend OnToology to a colleague
  20. Have you tried other similar software? In that case, which one and what do you think about it compared to OnToology?
  21. List the most negative aspect(s) about OnToology
  22. List the most positive aspect(s) about OnToology
