What is OnToology

A system to automate part of the collaborative ontology development process. Given a repository with an owl file, Ontoology will survey it and produce diagrams, a complete documentation and validation based on common pitfalls.

How to use it

  1. Enter your repo as user/repo i.e. myuser/myrepo (it does not work with organizations at the moment).
  2. Update your repo and push.
  3. If pull requests are created successfully you can find them in the pull requests pool with the diagram.
